== Description ==
Premium adult content platform theme with payment integration for PayPal, CashApp, Chime, and Crypto.

== Installation ==
1. Upload the theme folder to /wp-content/themes/
2. Activate the theme through Appearance > Themes
3. Install required plugins:
   - WPBakery Page Builder
   - Advanced Custom Fields
4. Configure payment methods in Theme Settings

== Features ==
- Model dashboard for content management
- Seller dashboard for payment tracking
- Manual payment verification system
- Mobile-responsive design
- WP Bakery Page Builder integration
- Custom user roles (Model, Subscriber, Seller)
